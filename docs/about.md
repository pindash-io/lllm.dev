# About lllm.dev

lllm.dev covers the **local LLM ecosystem** — new model releases, tooling updates, benchmarks, and everything happening in on-device AI.

## Why local?

<div class="grid cards" markdown>

-   **:material-shield-check: Privacy**
    Your data never leaves your machine.

-   **:material-cash-remove: No API costs**
    Run models as much as you want, no per-token billing.

-   **:material-wifi-off: Offline**
    Works without internet — on planes, in remote areas, anywhere.

-   **:material-lock: Control**
    No rate limits, no censorship, no vendor lock-in.

-   **:material-rocket-launch: Speed**
    Sub-10ms latency on local hardware — no network round-trips.

</div>
