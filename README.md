# lllm.dev

News on local LLMs — covering models, tools, and the open-source AI ecosystem.

**[lllm.dev](https://lllm.dev)**

## Stack

- **Zensical** (MkDocs fork) static site generator
- **TOML** config (`zensical.toml`)
- Markdown content in `docs/`

## Development

```bash
# Install
pip install zensical

# Serve (build + live reload)
zensical serve

# Build only
zensical build
```

Site runs at `http://localhost:8001`.
